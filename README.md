# CardioX

> Suplemento cardiovascular con omega-3 EPA/DHA, coenzima Q10 y ajo envejecido para la salud del corazón.

**[Visitar cardiox.pe](https://cardiox.pe)**

## Sobre este proyecto

Este repositorio contiene el sitio web de [CardioX](https://cardiox.pe), un recurso de salud basado en evidencia científica enfocado en **Salud Cardiovascular**.

### Ingredientes clave

Omega-3 (EPA 600mg + DHA 400mg), coenzima Q10 (100mg), ajo envejecido, vitamina K2

## Tecnología

- HTML5 + CSS3 (sitio estático)
- Hospedado en GitHub Pages
- Sin dependencias de JavaScript frameworks
- Optimizado para Core Web Vitals

## Evidencia científica

Todo el contenido está respaldado por:
- Estudios clínicos publicados en PubMed
- Monografías de la EMA (European Medicines Agency)
- Guías de la OMS sobre plantas medicinales
- Base de datos TRAMIL de plantas del Caribe y Latinoamérica
